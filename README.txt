Para correr la aplicación es necesario correr el archivo .main, que conectará a la base de datos del 
proyecto, y debería mostrar el fronting la página creada en html, pero al momento de intentar probar el
código realizado, la base de datos estaba caída. 

A partir de la lógica, se crean las tablas con cada una de las clases definidas, con sus PK, para las clases
que necesitan un PK generada, se creó una tabla aparte en el que se cumpla la conjunción de PK's.

Se debería acceder a los botones de las opciones principales y de allí poder realizar las consultas. (Todo está 
especificado en el código, aunque es posible que se muestre un error al no haber podido probarlo por las bases de datos)
